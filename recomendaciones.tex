\begin{itemize}
	\item Implementar un protocolo de salida de datos tipo cliente, lo que facilitaría la trasmisión de datos. Esto debido a que con el protocolo implementado establecer la conexión con la red depende de conocer la dirección IP del nodo central.
	\item Para mejorar los tiempos de respuesta del nodo con bus serial se recomienda agregar una opción en la configuración para introducir los esclavos asociados a este, si se suministra ID, registros y toda la información del esclavo se podría extraer y almacenar la data de manera continua y no tener asociado el retraso de esperar la respuesta cada vez que tenga una trama.
	\item Mejorar los tiempos de respuesta de la página de configuración del usuario aligerando imágenes y optimizando el código de los formularios del mismo, también se podría considerar realizarlos con otra tecnología web.
	\item Mejorar los tiempos de respuesta de la red utilizando tramas más cortas que las modbus mediante el desarrollo de un protocolo propio o utilizando un protocolo más orientado al internet de las cosas como MQTT.
\end{itemize}
